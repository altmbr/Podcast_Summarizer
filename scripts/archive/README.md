# Archived Scripts

This folder contains one-time utility scripts that have already been executed or are no longer actively used.

## Migration/Standardization Scripts (Executed)

- `fix_duplicate_timestamp_links.py` - Fixed duplicate timestamp links in summaries
- `standardize_all_summaries.py` - Standardized summary format across all episodes
- `update_summary_timestamps.py` - Updated timestamp formatting in summaries
- `backfill_dates.py` - Backfilled publication dates for old episodes

## Utility Scripts (Rarely Needed)

- `reset_episode_status.py` - Resets episode status (dangerous - use with caution)

## Image Generation Scripts (One-Time Use)

- `generate_og_image.py` - Generated initial OG image (superseded by v2)
- `generate_hero_no_text.py` - Generated hero image variant
- `generate_favicon.py` - Generated favicon (already created)

## Conversion Scripts (Deprecated)

- `convert_to_pdf.py` - PDF conversion utility
- `convert_md_to_pdf.py` - Markdown to PDF converter
- `create_summaries_html.py` - HTML summary generation
- `create_summaries_pdf.py` - PDF summary generation
- `summarize_single_episode.py` - Single episode summarization (use main script instead)

---

**Note:** These scripts are kept for reference but are not part of the active workflow.
If you need to re-run any of these, move them back to the root directory.
